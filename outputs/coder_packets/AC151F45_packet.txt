- (x - 3)(x + 1) = 0

## Step 2: Determining Which Function is on Top
I checked a point between x = -1 and x = 3 to see which function gives larger y-values.

At x = 0:
- The parabola: y = 0² = 0
- The line: y = 2(0) + 3 = 3

## Step 3: Setting Up the Integral
The area is:
$$A = \int_{-1}^{3} [(2x + 3) - x²] \, dx$$

## Step 4: Evaluating the Integral
$$A = \int_{-1}^{3} (2x + 3 - x²) \, dx$$

$$= \left[x² + 3x - \frac{x³}{3}\right]_{-1}^{3}$$

At x = -1: 1 - 3 + 1/3 = -5/3

$$A = 9 - (-5/3) = 9 + 5/3 = 27/3 + 5/3 = \frac{32}{3}$$

**The area is 32/3 square units.**
